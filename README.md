# DVC Pond Research

Website for the **Diablo Valley College (DVC) Pond Research** project—a **MESA**-run research program studying pond habitats on campus and how we can make them more habitable for the organisms that live there.

## Bio team

Brenda, Ariel, Ishak, Rita, and Preston.

## Program

- **Host:** DVC **MESA** (Mathematics, Engineering, Science Achievement)
- **Focus:** Field research across campus pond sites (Ponds **0–4**), documenting methods, results, and discoveries—see [`PLAN.md`](./PLAN.md) for site structure and goals.

## Repository

This repo holds the project website source. Styling prep includes Google Fonts in [`styles/fonts.css`](./styles/fonts.css).
